# ODSV2

Site vitrine premium pour ODS Group (chorégraphie, danse, spectacles).

## Lancer en local

```bash
python3 -m http.server 8080
```

Puis ouvrir `http://localhost:8080`.

## Pages

- Accueil
- À propos
- Services
- Réalisations
- Équipe
- Médias
- Contact

## Notes

- Le site contient des médias fictifs liés au secteur (Unsplash + YouTube) à remplacer par les contenus définitifs.
- Guide de remplacement: `assets/media/README.md`.
